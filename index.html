偵測中...

💡 排除此問題的 3 個黃金檢查步驟：

  1. 啟用「匿名登入」驗證： 到 Firebase 控制台 -> 左側選單 Authentication -> 點擊 Sign-in method 頁籤 -> 點選新增提供者 -> 選擇 Anonymous (匿名) 並啟用儲存。
  2. 建立「Firestore 資料庫」並開放規則： 到左側選單 Firestore Database -> 點擊「建立資料庫」 -> 選擇測試模式啟動 (允許讀寫)。
  3. 關閉 AdBlock 或 Brave 阻擋： 某些廣告/隱私阻擋器會阻擋 Firebase 的驗證網域，請暫時將其針對您的 GitHub Pages 關閉。
